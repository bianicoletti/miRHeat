# miRHeat: Ferramentas para análise de interações miRNA–UTR

O pacote miRHeat fornece funções para ler arquivos de predição de
interação entre genes e miRNAs, selecionar métricas de score, preparar
dados e gerar heatmaps de interações miRNA × UTR.
